# Solving Systems of Equations

A system of equations is a set of equations that share variables. The solution to the system is where the two graphs have a point of intersection ( Where thye cross.)

## Example
>y=2(x-3)^2-5
>>x^2+y=13

Solutions:
(0,13), (4,-3)
